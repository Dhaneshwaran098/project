# Student Permission Portal

A Flask-based web application for managing student leave requests with
role-based access for Students, Instructors, and HOD.

## Features
- Student leave request submission with file upload
- Instructor approval / rejection
- HOD final approval
- Role-based dashboards
- Session-based authentication
- Clean UI with modern styling

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- GitHub for version control

## How to Run
1. Install dependencies
2. Run `python app.py`
3. Open `http://127.0.0.1:5001`
